### 4.3航次库存看板
菜单位置：航次分组下，与航次列表、航次模板并列。

搜索筛选：
+ 航次号关键词搜索
+ 航次下拉筛选（列出已有库存数据的航次）

表格列（从左到右）：
+ 勾选框（支持全选/本页全选，选中后顶部出现批量操作按钮）
+ 航次号（等宽字体突出显示）
+ 船舶名称
+ 舱房类型（标签样式，如「套房」「阳台房」）
+ 总房量
+ 已售（绿色数字）
+ 锁定（红色数字，0 时灰色淡化）
+ 维修（黄色数字，0 时灰色淡化）
+ 可售（大字加粗，不足 5 间时红色预警）
+ 操作列（当前锁定状态显示「解锁」或「锁定」按钮）

可售计算公式：总房量 − 已售 − 锁定 − 维修

批量操作：
+ 勾选数据后，顶部出现「锁定（N）」和「解锁（N）」按钮
+ 点击弹出确认弹窗 → 确认后批量更新
+ 单行也可直接点击操作列的锁定/解锁按钮

数据来源：每个航次 × 每种舱房类型 = 一行库存记录，由航次生成时自动创建。

### 4.4航次价格日历
入口：航次列表每行操作列「定价」按钮，点击弹出价格日历页面。**是独立页面。**

数据结构：每条记录 = 航次 + 舱房类型 + 日期，包含四个价格字段（基准价、成人价、儿童价、婴儿价）。

弹窗布局（日历矩阵）：
+ 标题栏显示「价格日历 · 航次号」
+ 顶部切换条：成人价 / 儿童价 / 婴儿价 / 基准价（四个按钮，选中高亮），切换后整个矩阵显示对应价格类型
+ 表左侧固定列：舱房类型列表
+ 表右侧可滚动列：启航日期所在月份的全部日期（月历完整列，28-31天），航次日期列头深色可编辑，非航次日期列头浅灰置灰不可编辑
+ 每个格子：鼠标click日的格子后时进入编辑弹窗
+ 弹窗交互：列是舱房类型，行是票种类型，每个格子是一个可编辑的价格
+ 批量设置：点击后弹出批量设置弹窗
+ 批量设置弹窗字段：
- 生效日期*（日期段选择器）
- 生效星期*（复选框，周一-周日）
- 表格：该航线+模板下的开航每一天的价格（月-日格式，从取今天前后一个月的那种日历样式）

交互：
+ 切换价格类型后，所有格子即时刷新为对应价格
+ 修改任意格子后点击底部「保存定价」统一提交
+ 底部显示统计：「3种舱房 × 5天 · 顶部切换价格类型」

数据来源：航次生成时，从产品定价 × 票类规则自动计算初始值（如套房基准价 800、阳台房 500、海景房 300，儿童价为基准价的 30%，婴儿价为 10%）。每个格子都支持独立编辑。
