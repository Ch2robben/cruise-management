## 9.码头管理
码头从属于港口，一个港口可关联多个码头。

搜索筛选建议：
+ 无独立搜索页，码头数据在港口列表中通过树形表格展示

表格（树形表格）：
+ 父行：港口名称、英文名称、港口编码、城市、排序号、状态、操作
+ 子行：码头名称（前缀蓝色"码头"标签）、英文名称、排序号
+ 点击父行展开/收拢子行，子行左侧蓝色树形连线

表单交互：
+ 港口新增/编辑弹窗底部增加「码头管理」区域
+ 动态增减表格行：码头名称*、英文名称*、排序号
+ 默认一行，至少保留一行不被删除

详情展示：
+ 港口详情抽屉新增「码头列表」卡片，展示每个码头的名称/英文/排序

操作：
+ 与港口共用新增/编辑/删除，无独立 CRUD
