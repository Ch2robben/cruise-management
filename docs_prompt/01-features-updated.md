# 新增章节（4.3 航次库存看板 + 4.4 航次价格日历）

插入位置：## 4.2 航次模板 之后，## 5.游轮管理 之前

---
### 4.3航次库存看板
搜索筛选建议：

+ 关键词（航次号）
+ 航次（下拉单选，关联已有航次）

表格列建议：航次、船舶、舱房类型（Tag标签）、总房量、已售（绿色数字）、锁定（红色数字）、维修（黄色数字）、可售（大字号，<5间红色预警）

操作：
+ 单条锁定/解锁切换、批量勾选后一键锁定或解锁
+ 库存数据由航次生成时自动初始化（每种舱房类型一行），支持后续手动调整

数据来源：基于航次关联的船舶舱房类型自动生成库存行，初始值从船舶配置中读取。
---
### 4.4航次价格日历
交互入口：航次列表每行操作列「定价」按钮。

弹窗布局：日历矩阵

+ 顶部切换条：成人价 / 儿童价 / 婴儿价 / 基准价（SegmentControl 样式，切换后矩阵内所有格子显示对应价格类型）
+ 行 = 舱房类型（sticky 左侧固定）
+ 列 = 航次每一天（MM-DD，从开航日到终到日，可横向滚动）
+ 格子 = 数字输入框（hover 边框加深，focus ring 高亮）

交互：
+ 切换价格类型后整个矩阵即时刷新数据
+ 修改任意格子后点击「保存定价」统一提交
+ 底部显示「X种舱房 × Y天」统计信息

数据来源：航次生成时从产品基准价 × 票类规则自动计算初始值，价格数据关联航次ID + 舱房类型 + 日期，支持每个格子独立编辑。
