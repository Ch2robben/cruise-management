# 游轮需求文档索引

## 1. 长江游轮酒店运营组数智化项目需求说明书
- **来源**：酒店运营组（2026.02.26）
- **游轮相关度**：⭐⭐⭐⭐⭐ 100% 游轮
- **核心内容**：
  - 餐饮管理：菜单技术规格书与采购单品联动、餐料采购/领用/库存、厨余称重管理
  - 数据应用：物资消耗分析、采购价格数据库、二消产品实时监控
  - 客房智能控制：语音指令、能耗物资精准管理
  - AI智旅伴游：数字人导览、实时导航、应急指导
  - 自动分团→分桌→分车全流程自动化
- 🔗 [打开文档](https://www.feishu.cn/wiki/P5ABw1jdlifOehkfkjWczoASnEb)

## 2. 长航文旅一体化平台需求说明书（产品中心 0409）
- **来源**：产品中心（2026.04.09）
- **游轮相关度**：⭐⭐⭐⭐ 覆盖游轮产品全链条
- **核心内容**：
  - 数据管理：游轮运营数据（航线/航次排班/舱位/船舶负载）、长江游轮行业大盘数据采集
  - 产品管理：产品要素查询（船舶上下水航行时间、景区信息）、运营工单派发至长江游轮
  - 客户服务：在线选舱、值船、在船一站式服务
  - AI模块：智能行程规划、自动组合产品
- 🔗 [打开文档](https://www.feishu.cn/wiki/Y0w5wKkq4iwIjMkyektcS5Ccn0e)

## 3. 长航文旅销售中心一体化平台系统建设需求（0409）
- **来源**：销售中心（2026.04.09）
- **游轮相关度**：⭐⭐⭐⭐ 舱位/床位管理
- **核心内容**：
  - 舱位管理：床位库存与额定载客双库存管理、多级库存预警
  - 价格管理：游轮产品价格多维度批量上架
  - 数据管理：游轮产品数据、游客画像（不同游轮分析）
  - 跨系统对接：与游轮管理系统实时同步（舱位/价格/库存）
- 🔗 [打开文档](https://www.feishu.cn/wiki/GcMZwHy7di0Ig2ktgzvctxUgnob)

## 4. 长航文旅一体化运营平台建设需求书（财务部 0228）
- **来源**：财务部（2026.02.28）
- **游轮相关度**：⭐⭐⭐⭐ 明确区分长江游轮/城市游船
- **核心内容**：
  - 长江游轮：OTA渠道自动对账、航次完成自动收入确认、现金收款管理
  - 成本分摊：按舱位占比、按航次等规则分摊公共成本
  - 业财分析：单船单航次损益表、人均对客成本、航次航行成本
- 🔗 [打开文档](https://www.feishu.cn/wiki/BSO6wwsZUi0rg3kcQwScP6ygnCy)
