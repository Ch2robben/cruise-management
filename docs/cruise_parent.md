# 🚢 游轮·跨城航行

> **游轮业务定义**：跨城航行 + 船上住宿。区别于城市游船（同城观光、无住宿、无过夜）。

本目录从「长航-财务」知识空间 20 个文档中，逐篇阅读内容后，梳理出与**长江游轮**直接相关的文档。

核心判断标准：
- 文档中是否涉及「航次」「舱位/舱房」「二消」「岸上游」「游轮酒店」「单船单航次」等游轮专属概念
- 财务部需求书中明确区分了「长江游轮」和「城市游船」两种业态
