## 关联：财务部需求书中游轮相关成本指标

从财务部需求书（0228 & 0319 版）中提取的游轮专属财务指标：

| 指标 | 说明 |
|------|------|
| 单船单航次收入/成本/毛利/利润 | 按艘船 × 航次维度的损益分析 |
| 人均对客成本 | 人均天餐料成本、人均天客用品成本 |
| 航次航行成本 | 单航次航行直接成本 |
| 成本分摊规则 | 按舱位占比、按消费额、按人数、按航次 |
